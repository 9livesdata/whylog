class whylog.assistant.regex_assistant.regex_match.RegexMatch(line_object)[source]¶

Bases: object

Container for information about line and corresponding regex Keeps own data integrity - especially consistency between regex and groups (param_groups). Verifies updates.

Parameters:
  • line_text – raw string of the line
  • param_groups (dict[int, ParamGroup]) – represents params from text (catched by regex groups)
  • regex – regex matching to line_text
  • guessed_pattern_matches (dict[int, PatternMatch]) – keeps guessed regexes that match to line_text
